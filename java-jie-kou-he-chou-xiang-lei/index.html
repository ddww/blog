Java 接口和抽象类

2020-07-07

什么是接口？

接口通常以interface来修饰，是对行为的抽象，类通过继承接口的方式，继承接口的抽象方法。

特点

  • 接口的修饰符可使用public修饰，不加默认是default
  • 接口中的属性隐式的使用public static final修饰
  • JDK1.8 接口中的方法包含三种
    • 默认方法（default修饰）
    • 静态方法（public static修饰）
    • 抽象方法（public abstract修饰，我们定义的接口没有抽象方法或是不实现，将毫无意义）
  • 可实现接口多继承
  • 接口不能实例化对象
  • 接口没有构造方法
  • 接口不是被类继承了，而是要被类实现
